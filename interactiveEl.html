Interactive Elements

In-text link

CSS selector:

.in-text-link

Sample code:

<p>this is a <a class="in-text-link" href="#">test</a></p>

Rendered element:

this is a test

Navigation Link

CSS selector:

.button-nav

Sample code:

<a href="#" class="button-nav">Default Button</a>

Rendered element:

Default Button

Submit/Reset Buttons

CSS Selector:

.button-submit

Sample Code:

<input type = "submit" name = "submit" value = "Submit>
<input type = "reset" name = "reset" value = "Reset">

Rendered element:

Text Input Field with Label

CSS Selector:

.text-area
.label

Sample code:

<label for = "input-name">Name:</label>
<input type = "text" id = "input-name" placeholder = "Sam Black">
