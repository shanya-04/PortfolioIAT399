Combined Elements

Project Card

CSS Selectors:

.card · .card-img-placeholder / .card-img · .card-body · .card-tag · .card-title · .card-desc

Elements Combined:

Image as a link + heading (h3) + paragraph + small uppercase tag text

Sample Code:

<article class="card"> <a class="img-as-link" href="project.html"> <img class="card-img" src="img/project.jpg" alt="Project screenshot"> </a> <div class="card-body"> <p class="card-tag">UX / UI Design</p> <h3 class="card-title">Project Title</h3> <p class="card-desc">Short project description.</p> </div> </article>

Rendered Element:

UX / UI Design

Project Title

Short description of the project — what it is, what problem it solves, and your role.

Navigation Link with Label

CSS Selectors:

.button-nav · p (body text)

Elements Combined:

Navigation link button + descriptive paragraph text

Sample Code:

<p>Ready to see the work?</p> <a class="button-nav" href="projects.html">View Projects</a>

Rendered Element:

Ready to see the work?

View Projects

In-Text Link Inside Paragraph

CSS Selectors:

p · .in-text-link

Elements Combined:

Body paragraph + inline anchor link

Sample Code:

<p> I'm currently studying at <a class="in-text-link" href="https://www.sfu.ca">Simon Fraser University</a>, where I focus on interactive design and development. </p>

Rendered Element:

I'm currently studying at Simon Fraser University, where I focus on interactive design and development.

Labelled Input Field

CSS Selectors:

.label · .text-area

Elements Combined:

Form label + text input, paired as a single accessible field unit

Sample Code:

<label class="label" for="input-name">Your Name</label> <input class="text-area" type="text" id="input-name" placeholder="Sam Black"> <label class="label" for="input-email">Email Address</label> <input class="text-area" type="email" id="input-email" placeholder="sam@example.com">

Rendered Element:

Contact Form

CSS Selectors:

.label · .text-area · .button-submit

Elements Combined:

Labelled inputs + submit and reset buttons — a full minimal form unit

Sample Code:

<label class="label" for="msg-name">Name</label> <input class="text-area" type="text" id="msg-name" placeholder="Your name"> <label class="label" for="msg-body">Message</label> <input class="text-area" type="text" id="msg-body" placeholder="Your message"> <input class="button-submit" type="submit" value="Send Message"> <input class="button-submit" type="reset" value="Clear">

Rendered Element:

Section Header Block

CSS Selectors:

.section-label · h2 (.section-title) · p

Elements Combined:

Small uppercase label + display heading + body paragraph — the opening block of every content section

Sample Code:

<p class="section-label">Selected Work</p> <h2 class="section-title">Projects</h2> <p>A selection of recent work across design and development.</p>

Rendered Element:

Selected Work

Projects

A selection of recent work across design and development.

Image Link with Caption

CSS Selectors:

.img-as-link · p

Elements Combined:

Clickable image wrapped in an anchor + a supporting caption paragraph

Sample Code:

<a class="img-as-link" href="project.html"> <img src="img/project-thumb.jpg" width="300" height="200" alt="Project preview"> </a> <p>Project title — UX/UI Design, 2025</p>

Rendered Element:

Project Title — UX/UI Design, 2025
