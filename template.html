Template Structure

Key Sections

  1. <nav class="site-nav"> — fixed top navigation bar
  2. <section class="hero"> — full-height hero with eyebrow, h1, subtitle, CTAs
  3. <section class="section"> — reusable content section with label + title
  4. <div class="cards-grid"> — responsive card grid for projects
  5. <div class="about-strip"> — two-column about/intro strip
  6. <footer class="site-footer"> — footer with copyright and links

Available Classes

  1. .btn-primary — filled dark button
  2. .btn-ghost — outline button
  3. .fade-up — entrance animation (add to hero children)
  4. .section-label — small uppercase terracotta label
  5. .section-title — display-font section heading
  6. .card — project card with hover lift
